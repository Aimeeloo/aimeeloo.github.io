Hexo 架設自己的 Blog

在正式開始架設自己的Blog之前呢
首先環境設定必需要有
Node.js ( 開啟terminal可以利用node -v查看版本或是否安裝
Git

Step01 先下載 Hexo

開始透過npm安裝Hexo，在terminal中輸入安裝指令：

Step02 建立資料夾 並且初始化

第一個命令 aimee_blog 是自己的資料夾名稱，請由個人喜好自取喔！
再來請依照順序完成以下命令。

完成之後，專案開啟目錄可以看到以下結果：

_config.yml： 網站 配置 檔案，可以在此配置大部分的設定。
package.json： 套件版本控制
scaffolds： 鷹架 資料夾。建立新文章時，會根據 scaffold 來建立檔案。
source： 原始檔案資料夾是放置內容的地方。
themes： 主題 資料夾。Hexo 會根據主題來產生靜態檔案。

並且啟用 hexo-sever

如果有錯誤可以參考官方文件 https://hexo.io/zh-tw/docs/server.html

Step05 安裝Git

Step06 將網站佈署到伺服器

hexo deploy (hexo d) 部署網站。
hexo generate (hexo g) 產生靜態檔案，會在目錄下產生 public 資料夾。

Step07 設定repository

在 GitHub 剛建立的 Repository，並且進入 setting 頁面
並且下拉到 GitHub Pages，點選綠色框框裡的網址就可以成功連結到你的blog！

Step08 更換Blog樣式主題

可以進入連結 https://hexo.io/themes/ 套用樣式

並且點選藍色大標，會連結到 Github 文件，並且按照文件安裝，就可以幫 Blog 套版囉！
