Click 點擊觸發

v-on:click="onClick"
onClick 在這是一個方法，透過方法來指定點選之後的動作。
若有需要參數，亦可傳參數 onClick(a,b)。

Mouse hover

onEnter、onLeave 當游標移動到按鈕時，
在 console 就會出現 Mouse Enter；游標離開就會出現 Mouse Leave。
在 function 裡的 this.msg = '...' 就會改變原本 data 裡的 msg 內容。

這裡就先用 codepen 看效果吧！

Submit

這時送出會發現畫面已重新整理，是因為瀏覽器會發送 get 或是 post 請求到指定的頁面
為了避免此情況發生，請參照以下：

preventDefault() 方法，是用來阻止元素發生默認行為。

onSubmit($event) 在這使用 $event 這個關鍵詞 Vue 就會知道你要傳入的是一個事件，
此時就可以正常送出，不會再刷新。

然後也可以使用 prevent 方法來寫，這樣做法與以上範例是相同的意思。

其他方法：
.stop - 等同於 event.stopPropagation()，防止事件冒泡。
.prevent - 等同於 event.preventDefault()，防止執行預設的行為。
.capture - 與事件冒泡的方向相反，事件捕獲 (event capturing) 是由外而內的。
.self - 只會觸發自己範圍內的事件，不包含子元素。
.once - 只會觸發一次

KeyUp

監聽鍵盤事件的特定鍵值，這裡用 .enter 為例。

當按 Enter 時，console 就會出現 Entered，其他則無反應。

其他特定鍵：
.enter .tab .delete( 含 delete 和 backspace 鍵 )
.esc .space .up .down .left .right
.ctrl .alt .shift .meta
