v-bind 屬性綁定，用來綁定數據與元素的屬性。
我們最常在HTML裡最長使用 <a> 標籤超連結，直接把連結位置寫死。
像是這樣：

如果我們希望者是一個動態地址呢？
這時我們就可以使用 Vue.js 的 v-bind 來綁定HTML屬性
寫法就會像以下這樣。

文字連結

將文字超連結位置倒向 https://a-min.art/ 頁面

開啟 Console 輸入 app.url = "網址位置" 可以更改網址位置。
此時重新整點選點我，網址位置就會倒向 https://www.google.com.tw/ 頁面

圖片連結

此時開啟網頁就會看到圖片，點下去就可以連結到指定的網站。

class 綁定

在 data 裡新增綁定的 class klass: 'btn btn-default'

在要綁定的標籤加上 v-bind:class="klass”

接著從開發者工具來看，會發現在 <a> 標籤上複雜的 v-bind:class 變成很單純的 class

加入 style CSS

在 main.js 加上 isActive: true

Index.html 把 <a> 標籤 class 的改成 v-bind:class="{active: isActive}”

開啟網頁，並且開啟 Console 查看， isActive 為 true 時，會顯示樣式設定的 css。

當 isActive 為 false 時，css 樣式就會消失。
